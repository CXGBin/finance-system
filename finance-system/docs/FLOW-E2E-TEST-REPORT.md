## 发现的问题

### BUG (需修复)

1. **税务日历DateTime解析异常** - `GET /api/tax/calendar?year=2026` 无month参数时，默认month=0导致DateTime异常
2. **数据库缺表** - 远程数据库缺少 `fm_asset_inventory`、`fm_asset_change`、`fm_expense_allocate` 三张表（已手动创建修复）

### 数据依赖问题 (测试脚本改进项)

3. 报表接口参数格式为 `Period=2026-01` 而非 `year/month`
4. 报表导出 `reportType` 需使用后端枚举值
5. 审批流程模板nodes格式需匹配后端校验规则
6. 发票创建 `InvoiceType` 需为int类型(1=进项/2=销项)
