# 模块三：报表中心 — 详细功能需求文档

> 版本：v1.0 | 日期：2026-06-15 | 状态：待确认  
> 模块标识：`report` | 默认状态：开启  
> 依赖模块：`account`（账务管理）

---

## 1. 模块定位

报表中心负责从账务模块的凭证和科目余额数据中自动生成各类财务报表，是财务管理系统的数据输出层。报表数据**完全由系统自动计算**，用户仅可选择查询条件和导出格式，不可手动修改报表数据。

---

## 2. 功能清单总览

| 序号 | 功能 | 说明 |
|------|------|------|
| R-01 | 资产负债表 | 自动生成，符合 CAS 格式 |
| R-02 | 利润表（损益表） | 自动生成，支持期间对比 |
| R-03 | 现金流量表 | 自动生成，支持直接法 |
| R-04 | 科目余额表 | 各科目余额汇总查询 |
| R-05 | 明细账表 | 各科目明细账导出 |
| R-06 | 自定义报表 | 用户自定义报表模板与生成 |
| R-07 | 报表导出 | Excel / PDF 导出 |
| R-08 | 多期对比分析 | 同一报表多期间对比展示 |

## 3. 详细功能说明

### R-01 资产负债表

#### 业务规则
- 依据期末科目余额自动生成
- 报表格式遵循《企业会计准则》资产负债表标准格式
- 资产 = 负债 + 所有者权益（强制校验平衡）
- 左侧资产项目、右侧负债及所有者权益项目
- 资产项目按流动性排列：流动资产 → 非流动资产
- 负债项目按偿还期排列：流动负债 → 非流动负债
- 每行数据标注对应的科目编码映射关系
- 支持「年初数」和「期末数」两列对比
- 支持查询任意历史期间的资产负债表
- 若科目余额表有数据，则强制平衡校验通过后才能生成

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| period | string | 会计期间（年-月），如 `2026-06` |
| show_zero | bool | 是否显示金额为零的行，默认否 |

#### 标准行项目映射（核心）

| 行次 | 报表项目 | 对应科目类型 | 取值方式 |
|------|---------|------------|---------|
| 1 | 货币资金 | 资产类-库存现金/银行存款/其他货币资金 | 借方余额合计 |
| 2 | 应收账款 | 资产类-应收账款 | 借方余额-坏账准备贷方余额 |
| 3 | 预付账款 | 资产类-预付账款 | 借方余额 |
| 4 | 存货 | 资产类-原材料/库存商品/周转材料/生产成本等 | 借方余额合计 |
| 5 | 固定资产 | 资产类-固定资产 | 借方余额-累计折旧贷方余额 |
| 6 | 无形资产 | 资产类-无形资产 | 借方余额-累计摊销贷方余额 |
| 7 | 短期借款 | 负债类-短期借款 | 贷方余额 |
| 8 | 应付账款 | 负债类-应付账款 | 贷方余额 |
| 9 | 应付职工薪酬 | 负债类-应付职工薪酬 | 贷方余额 |
| 10 | 应交税费 | 负债类-应交税费 | 贷方余额（借方余额以负数或 `-` 标注） |
| 11 | 长期借款 | 负债类-长期借款 | 贷方余额 |
| 12 | 实收资本 | 权益类-实收资本 | 贷方余额 |
| 13 | 资本公积 | 权益类-资本公积 | 贷方余额 |
| 14 | 盈余公积 | 权益类-盈余公积 | 贷方余额 |
| 15 | 未分配利润 | 权益类-利润分配 | 贷方余额±本年利润 |

> 完整行项目可在开发阶段参照 CAS 资产负债表模板配置，支持后台映射调整。

#### 输出结构

```json
{
  "reportDate": "2026-06-30",
  "companyName": "XX公司",
  "currency": "CNY",
  "unit": "元",
  "assets": [
    {
      "lineNo": 1,
      "itemName": "货币资金",
      "beginningBalance": 500000.00,
      "endingBalance": 620000.00,
      "subjectCodes": ["1001", "1002", "1012"]
    }
  ],
  "liabilities": [
    {
      "lineNo": 1,
      "itemName": "短期借款",
      "beginningBalance": 200000.00,
      "endingBalance": 200000.00,
      "subjectCodes": ["2001"]
    }
  ],
  "equity": [ ... ],
  "totalAssets": 1200000.00,
  "totalLiabilitiesAndEquity": 1200000.00
}
```

#### 接口
- `GET /api/report/balance-sheet` — 查询资产负债表

### R-02 利润表（损益表）

#### 业务规则
- 依据损益类科目的本期发生额自动生成
- 格式遵循 CAS 利润表标准格式
- 基本公式：营业收入 - 营业成本 - 税金及附加 - 期间费用 = 营业利润
- 营业利润 + 营业外收支 = 利润总额
- 利润总额 - 所得税费用 = 净利润
- 支持选择「本月数」或「本年累计数」
- 支持同比/环比对比（与上期、上年同期对比，计算增长率）
- 期间选择：单月 or 累计至某月

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| period | string | 会计期间 |
| data_type | string | `month`（本月）或 `cumulative`（本年累计） |
| compare_period | string | 对比期间（可选，用于同比分析） |
| show_zero | bool | 是否显示金额为零的行 |

#### 核心行项目映射

| 行次 | 报表项目 | 计算规则 |
|------|---------|---------|
| 1 | 营业收入 | 收入类科目贷方发生额合计 |
| 2 | 营业成本 | 费用类-主营业务成本借方发生额 |
| 3 | 税金及附加 | 费用类-税金及附加借方发生额 |
| 4 | 销售费用 | 费用类-销售费用借方发生额 |
| 5 | 管理费用 | 费用类-管理费用借方发生额 |
| 6 | 财务费用 | 费用类-财务费用借方发生额 |
| 7 | 营业利润 | 行1-行2-行3-行4-行5-行6 |
| 8 | 营业外收入 | 收入类-营业外收入贷方发生额 |
| 9 | 营业外支出 | 费用类-营业外支出借方发生额 |
| 10 | 利润总额 | 行7+行8-行9 |
| 11 | 所得税费用 | 费用类-所得税费用借方发生额 |
| 12 | 净利润 | 行10-行11 |

#### 接口
- `GET /api/report/income-statement` — 查询利润表

---

### R-03 现金流量表

#### 业务规则
- 采用直接法编制
- 分三大类活动：经营活动、投资活动、筹资活动
- 现金流入 - 现金流出 = 现金净增加额
- 数据来源：标记了「现金」或「银行」属性的科目的凭证分录
- 支持按科目维度分类汇总（需凭证分录补充现金流量项目标注）
- **第一版可采用简化方案**：仅按现金/银行科目的借贷发生额汇总

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| period | string | 会计期间 |
| data_type | string | `month` / `cumulative` |

#### 现金流量项目分类

| 活动 | 流入项目 | 流出项目 |
|------|---------|---------|
| 经营活动 | 销售商品收到的现金、税费返还 | 购买商品支付的现金、支付工资、支付税费 |
| 投资活动 | 收回投资、处置资产 | 购建固定资产、对外投资 |
| 筹资活动 | 吸收投资、借款 | 偿还借款、分配股利 |

#### 接口
- `GET /api/report/cash-flow` — 查询现金流量表

---

### R-04 科目余额表

#### 业务规则
- 展示所有科目在指定期间的余额和发生额
- 层级展示：支持展开到任意级次
- 金额精度统一 2 位小数
- 余额方向：借方余额显示在借方列，贷方余额显示在贷方列
- 末级科目余额合计 = 上级科目余额（自动校验）
- 支持按科目类型筛选（资产/负债/权益/收入/费用）
- 支持导出 Excel

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| period | string | 会计期间 |
| subject_type | int | 科目类型（可选） |
| level | int | 展示到第几级（可选） |
| show_zero | bool | 是否显示无发生额的科目 |

#### 输出字段

| 字段 | 类型 | 说明 |
|------|------|------|
| subject_code | string | 科目编码 |
| subject_name | string | 科目名称 |
| level | int | 级次 |
| begin_debit | decimal | 期初借方 |
| begin_credit | decimal | 期初贷方 |
| current_debit | decimal | 本期借方发生额 |
| current_credit | decimal | 本期贷方发生额 |
| end_debit | decimal | 期末借方 |
| end_credit | decimal | 期末贷方 |
| year_debit | decimal | 本年累计借方 |
| year_credit | decimal | 本年累计贷方 |

#### 接口
- `GET /api/report/subject-balance` — 查询科目余额表

---

### R-05 明细账表

#### 业务规则
- 与账务管理的明细账查询相同，但提供导出功能
- 支持导出为 Excel（带格式）
- 可选择导出范围（全部/筛选结果）

#### 接口
- `GET /api/report/ledger-detail-export` — 导出明细账

---

### R-06 自定义报表

#### 业务规则
- 用户可创建自定义报表模板
- 每个模板定义若干行，每行关联一个或多个科目
- 行与行之间支持简单计算公式（加减乘除、合计行）
- 保存模板后可随时按期间生成报表
- 预置常用模板：管理费用明细表、销售费用明细表等

#### 模板字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| template_name | string | 是 | 报表名称 |
| description | string | 否 | 描述 |
| rows | object[] | 是 | 报表行定义 |
| created_time | datetime | 自动 | 创建时间 |
| updated_time | datetime | 自动 | 更新时间 |

#### 行定义字段

| 字段 | 类型 | 说明 |
|------|------|------|
| line_no | int | 行号 |
| item_name | string | 行项目名称 |
| subject_ids | long[] | 关联科目ID列表 |
| calculate_type | string | 取值方式：balance(余额)/debit(借方发生)/credit(贷方发生) |
| formula | string | 计算公式（如 `R1+R2-R3`，R 代表行号） |
| indent | int | 缩进级次 |

#### 接口
- `GET /api/report/custom/template/list` — 模板列表
- `POST /api/report/custom/template` — 创建模板
- `PUT /api/report/custom/template/{id}` — 修改模板
- `DELETE /api/report/custom/template/{id}` — 删除模板
- `GET /api/report/custom/template/{id}/generate` — 按模板生成报表数据

---

### R-07 报表导出

#### 业务规则
- 所有报表均支持导出
- 导出格式：Excel (.xlsx) 和 PDF
- Excel 导出保持表格格式（列宽、合并单元格、边框）
- PDF 导出保持 A4 纸张格式，带公司抬头
- 支持批量导出（如一键导出三大主表）

#### 接口
- `GET /api/report/export/excel?type={reportType}&period={period}` — Excel 导出
- `GET /api/report/export/pdf?type={reportType}&period={period}` — PDF 导出

---

### R-08 多期对比分析

#### 业务规则
- 同一报表可选择 2-4 个期间进行对比
- 对比方式：并列展示 / 增长率计算 / 差异额计算
- 前端使用图表辅助展示（柱状图/折线图）
- 仅适用于利润表和科目余额表

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| type | string | 报表类型（income-statement/subject-balance） |
| periods | string[] | 最多4个期间，如 `["2026-01","2026-02","2026-03"]` |
| display_mode | string | `parallel`（并列）/ `growth_rate`（增长率）/ `difference`（差异额） |

#### 接口
- `GET /api/report/compare` — 多期对比

## 4. 页面设计要求

| 页面 | 路由 | 布局说明 |
|------|------|---------|
| 报表首页 | `/report` | 卡片入口（三大主表+科目余额+自定义） |
| 资产负债表 | `/report/balance-sheet` | 顶部条件栏（期间选择）+ 标准资产负债表布局（左右两栏） |
| 利润表 | `/report/income-statement` | 顶部条件栏（期间+本月/累计+对比）+ 利润表表格 |
| 现金流量表 | `/report/cash-flow` | 顶部条件栏 + 现金流量表（三大活动分组） |
| 科目余额表 | `/report/subject-balance` | 顶部条件栏 + 可展开折叠的科目余额表格 |
| 自定义报表列表 | `/report/custom` | 模板列表卡片/表格 |
| 自定义报表设计 | `/report/custom/edit/:id` | 行编辑器（行项目+科目选择+公式输入） |
| 自定义报表查看 | `/report/custom/view/:id` | 生成的报表数据表格 |
| 多期对比 | `/report/compare` | 报表类型选择 + 期间多选 + 对比表格/图表 |

### UI 规范
- 报表金额列右对齐，千分位分隔符（如 `1,234,567.89`）
- 负数以红色显示或加括号 `(1,234.56)`
- 合计行/小计行加粗显示
- 导出按钮固定在页面右上角
- 报表数据为空时显示空状态提示
