## 一、项目概述

### 1.1 项目目标

开发一套前后端分离的企业级财务管理系统，支持模块化功能开关，可按需启用或关闭业务模块，满足不同规模企业的财务管理需求。

### 1.2 技术选型

| 层级 | 技术方案 | 说明 |
|------|---------|------|
| 前端框架 | React 18 + TypeScript | 前端 UI 框架 |
| UI 组件库 | Ant Design 5.x | 企业级 UI 组件 |
| 构建工具 | Vite 5.x | 快速构建与热更新 |
| 状态管理 | Zustand | 轻量级状态管理 |
| 路由 | React Router v6 | 前端路由管理 |
| HTTP 客户端 | Axios | API 请求 |
| 后端框架 | .NET 8 (ASP.NET Core) | 后端 API 框架 |
| ORM | SqlSugarCore | 轻量级 ORM |
| 数据库 | SQL Server 2019+ | 关系型数据库（单租户） |
| 依赖注入 | .NET 原生 DI + Autofac | 模块化 DI 管理 |
| 缓存 | Redis | 分布式缓存 |
| 文档 | Swagger / Knife4j | API 文档 |
| 日志 | Serilog | 结构化日志 |

### 1.3 架构原则

- **前后端分离**：前端通过 RESTful API 与后端交互，独立部署
- **模块化设计**：每个业务模块独立，通过开关控制启用/禁用
- **依赖注入**：所有服务通过 DI 容器管理，模块按需注册
- **分层架构**：Controller → Service → Repository，职责清晰
- **统一响应**：标准化 API 响应格式，统一错误处理

## 四、模块开关机制

### 4.1 设计思路

#### 4.2.3 模块注册中心

```csharp
// Infrastructure/DI/ModuleRegistry.cs
public class ModuleRegistry
{
    private readonly List<IAppModule> _modules = new();
    
    public void RegisterModule(IAppModule module)
    {
        _modules.Add(module);
    }
    
    public IReadOnlyList<IAppModule> GetAllModules() => _modules.AsReadOnly();
    
    public bool IsModuleEnabled(string moduleId)
    {
        var module = _modules.FirstOrDefault(m => m.ModuleId == moduleId);
        return module?.IsEnabled ?? false;
    }
    
    public void EnableModule(string moduleId)
    {
        var module = _modules.FirstOrDefault(m => m.ModuleId == moduleId);
        if (module != null) module.IsEnabled = true;
    }
    
    public void DisableModule(string moduleId)
    {
        var module = _modules.FirstOrDefault(m => m.ModuleId == moduleId);
        if (module != null && !module.IsCore)
            module.IsEnabled = false;
    }
    
    /// <summary>
    /// 仅注册已启用模块的服务到 DI 容器
    /// </summary>
    public void RegisterEnabledModules(
        IServiceCollection services, IConfiguration config)
    {
        foreach (var module in _modules.Where(m => m.IsEnabled))
        {
            module.RegisterServices(services, config);
        }
    }
}
```

#### 4.2.5 模块开关中间件

```csharp
// Infrastructure/Middleware/ModuleEnabledMiddleware.cs
public class ModuleEnabledMiddleware
{
    private readonly RequestDelegate _next;
    private readonly ModuleRegistry _moduleRegistry;
    
    public ModuleEnabledMiddleware(
        RequestDelegate next, ModuleRegistry moduleRegistry)
    {
        _next = next;
        _moduleRegistry = moduleRegistry;
    }
    
    public async Task InvokeAsync(HttpContext context)
    {
        var path = context.Request.Path.Value?.ToLower() ?? "";
        
        // 解析请求路径对应的模块标识
        var moduleId = ResolveModuleIdFromPath(path);
        
        if (moduleId != null && !_moduleRegistry.IsModuleEnabled(moduleId))
        {
            context.Response.StatusCode = StatusCodes.Status403Forbidden;
            await context.Response.WriteAsJsonAsync(new
            {
                code = 403,
                message = $"模块 [{moduleId}] 未启用，请联系管理员开启"
            });
            return;
        }
        
        await _next(context);
    }
    
    private string? ResolveModuleIdFromPath(string path)
    {
        // /api/account/xxx → account
        // /api/report/xxx  → report
        var segments = path.Split('/', StringSplitOptions.RemoveEmptyEntries);
        if (segments.Length >= 2 && segments[0] == "api")
            return segments[1];
        return null;
    }
}
```

### 4.3 前端模块开关

#### 4.3.1 模块状态管理

```typescript
// store/useModuleStore.ts
import { create } from 'zustand';

interface ModuleState {
  modules: {
    moduleId: string;
    moduleName: string;
    isEnabled: boolean;
    isCore: boolean;
  }[];
  enabledModules: string[];
  isModuleEnabled: (moduleId: string) => boolean;
  setModules: (modules: ModuleState['modules']) => void;
}

export const useModuleStore = create<ModuleState>((set, get) => ({
  modules: [],
  enabledModules: [],
  
  isModuleEnabled: (moduleId: string) => {
    return get().enabledModules.includes(moduleId);
  },
  
  setModules: (modules) => {
    set({
      modules,
      enabledModules: modules
        .filter((m) => m.isEnabled)
        .map((m) => m.moduleId),
    });
  },
}));
```

### 7.1 依赖注入设计

```csharp
// Infrastructure/DI/ServiceCollectionExtensions.cs
public static class ServiceCollectionExtensions
{
    public static IServiceCollection AddFinanceSystem(
        this IServiceCollection services, IConfiguration configuration)
    {
        // 1. 注册基础设施
        services.AddSingleton<ModuleRegistry>();
        services.AddSqlSugarSetup(configuration);
        services.AddJwtAuth(configuration);
        services.AddRedisCache(configuration);
        services.AddSerilogLogging();
        
        // 2. 注册所有模块定义（不论开关）
        var moduleRegistry = services.BuildServiceProvider()
            .GetRequiredService<ModuleRegistry>();
        RegisterAllModules(moduleRegistry, configuration);
        
        // 3. 仅注册已启用模块的服务
        services.AddEnabledModuleServices(moduleRegistry, configuration);
        
        // 4. 注册通用服务（不受模块开关影响）
        services.AddScoped(typeof(IBaseRepository<>), typeof(BaseRepository<>));
        
        return services;
    }
    
    private static void RegisterAllModules(
        ModuleRegistry registry, IConfiguration config)
    {
        // 从配置读取模块状态
        var moduleConfigs = config
            .GetSection("ModuleConfig:Modules").Get<List<ModuleConfigDto>>();
        
        // 注册所有模块实例
        registry.RegisterModule(new SystemModule());
        registry.RegisterModule(new AccountModule());
        registry.RegisterModule(new ReportModule());
        registry.RegisterModule(new BudgetModule());
        registry.RegisterModule(new ApprovalModule());
        registry.RegisterModule(new AssetModule());
        registry.RegisterModule(new ExpenseModule());
        registry.RegisterModule(new TaxModule());
        
        // 同步配置中的开关状态
        foreach (var mc in moduleConfigs ?? Enumerable.Empty<ModuleConfigDto>())
        {
            var module = registry.GetAllModules()
                .FirstOrDefault(m => m.ModuleId == mc.ModuleId);
            if (module != null)
                module.IsEnabled = mc.IsEnabled;
        }
    }
    
    private static void AddEnabledModuleServices(
        IServiceCollection services, ModuleRegistry registry,
        IConfiguration config)
    {
        foreach (var module in registry.GetAllModules()
            .Where(m => m.IsEnabled))
        {
            module.RegisterServices(services, config);
        }
    }
}
```
