# 模块八：税务管理 — 详细功能需求文档

> 版本：v1.0 | 日期：2026-06-15 | 状态：待确认  
> 模块标识：`tax` | 默认状态：开启  
> 依赖模块：`account`（账务管理，税额数据来源）

---

## 1. 模块定位

税务管理模块提供税种维护、纳税申报计算、发票管理和税务报表功能。通过从账务模块获取相关科目数据，自动计算应纳税额，辅助企业完成日常税务管理。本模块基于中国大陆税制（CAS），暂不涉及国际化税务。

## 3. 详细功能说明

### T-01 税种维护

#### 业务规则
- 预置中国大陆常见税种
- 每个税种配置税率、计税方式、申报周期、关联会计科目
- 支持多级税率（如增值税：一般税率13%、9%、6%、0%）
- 税率可按有效期配置（支持历史税率查询）
- 已使用的税率不可删除

#### 预置税种

| 税种 | 默认税率 | 计税方式 | 申报周期 |
|------|---------|---------|---------|
| 增值税 | 13%/9%/6%/0% | 销项-进项 | 月度 |
| 企业所得税 | 25%（小微企业5%/10%） | 应纳税所得额 | 季度 |
| 个人所得税（工资） | 3%-45% 累进 | 工资薪金 | 月度 |
| 附加税（城建税） | 7%/5%/1% | 增值税×税率 | 月度 |
| 印花税 | 按类别0.005%-0.05% | 按凭证金额 | 按次/月度 |
| 房产税 | 1.2% | 房产原值×(1-扣除率)×税率 | 半年度 |
| 土地使用税 | 按地区等级 | 占地面积×单位税额 | 半年度 |

#### 税种字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| tax_name | string | 是 | 税种名称 |
| tax_code | string | 是 | 税种编码（唯一） |
| description | string | 否 | 说明 |
| calculation_method | string | 是 | 计算方式描述 |

| related_subject_credit | bigint FK | 否 | 关联贷方科目 |
| status | tinyint | 是 | 0停用 1启用 |

#### 税率字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| tax_id | bigint FK | 是 | 税种ID |
| rate_name | string | 是 | 税率名称（如"一般税率"） |
| rate_value | decimal(5,4) | 是 | 税率值（如 0.13 表示 13%） |
| effective_start | date | 是 | 生效起始日期 |

### T-02 纳税申报计算

#### 业务规则
- 根据账务模块的凭证数据自动计算各项应纳税额
- **增值税**：
  - 销项税额 = 收入类科目（含税）对应的销项税额科目贷方发生额
  - 进项税额 = 进项税额科目借方发生额（可抵扣部分）
  - 应纳税额 = 销项税额 - 进项税额（≥0）
  - 进项税额转出计入当期
- **企业所得税**：
  - 利润总额 = 利润表净利润
  - 纳税调整 = 会计利润 ± 纳税调整事项
  - 应纳税所得额 = 利润总额 + 纳税调整
  - 应纳税额 = 应纳税所得额 × 税率
- **附加税**：
  - 城建税 = 增值税应纳税额 × 城建税率（市区7%、县城5%、其他1%）
  - 教育费附加 = 增值税应纳税额 × 3%
  - 地方教育附加 = 增值税应纳税额 × 2%
- 计算结果仅供参考，最终以税务机关核定为准
- 支持手工调整计算结果

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| tax_type_id | long | 税种 |
| period | string | 申报期间（年-月或年-季度） |

#### 接口
- `POST /api/tax/declare/calculate` — 计算应纳税额
- `GET /api/tax/declare/result/{declarationId}` — 查看计算结果

### T-03 纳税申报表

#### 业务规则
- 基于计算结果生成标准格式的纳税申报表
- 支持增值税纳税申报表、企业所得税季度纳税申报表等
- 申报表数据可手工修改（系统计算的为预填数据）
- 申报状态：待申报 / 已申报 / 已缴款
- 申报后生成缴款凭证记录

#### 申报表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint PK | 主键 |
| declaration_no | varchar(50) UNIQUE | 申报单号 |
| tax_type_id | bigint FK | 税种 |
| period | string | 申报期间 |
| taxable_amount | decimal(18,2) | 计税金额/所得额 |
| tax_amount | decimal(18,2) | 应纳税额 |
| actual_tax_amount | decimal(18,2) | 实际缴税额（可调整） |
| status | int | 0待申报 1已申报 2已缴款 |
| declared_by | bigint FK | 申报人 |
| declared_time | datetime | 申报时间 |
| paid_by | bigint FK | 缴款人 |
| paid_time | datetime | 缴款时间 |

#### 接口
- `GET /api/tax/declaration/page` — 申报表分页
- `GET /api/tax/declaration/{id}` — 申报表详情
- `POST /api/tax/declaration` — 创建申报（从计算结果生成）
- `PUT /api/tax/declaration/{id}` | — 修改申报数据
- `POST /api/tax/declaration/{id}/submit` | — 提交申报
- `POST /api/tax/declaration/{id}/pay` | — 确认缴款
- `GET /api/tax/declaration/export` | — 导出申报表

### T-04 发票管理

#### 业务规则
- 管理企业的进项发票和销项发票
- **进项发票**（购入时收到）：
  - 登记发票信息：发票号、开票日期、销售方、金额、税额
  - 支持发票查验（与外部发票查验接口对接，第一版仅手工登记）
  - 进项发票用于增值税抵扣
  - 发票状态：待认证 / 已认证 / 不可抵扣
- **销项发票**（销售时开具）：
  - 登记已开具发票信息
  - 与收入凭证关联
  - 发票状态：已开具 / 已作废
- 发票统计：按月度统计进项/销项税额

#### 发票字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| invoice_type | int | 是 | 1进项 2销项 |
| invoice_code | string | 是 | 发票代码 |
| invoice_no | string | 是 | 发票号码 |
| invoice_date | date | 是 | 开票日期 |
| seller_name | string | 是 | 销方名称 |
| buyer_name | string | 是 | 购方名称 |
| amount | decimal(18,2) | 是 | 金额（不含税） |
| tax_rate | decimal(5,4) | 是 | 税率 |
| tax_amount | decimal(18,2) | 是 | 税额 |
| total_amount | decimal(18,2) | 是 | 价税合计 |
| status | int | 是 | 进项：0待认证 1已认证 2不可抵扣；销项：0已开具 1已作废 |
| voucher_id | bigint FK | 否 | 关联凭证ID |
| remark | varchar(500) | 否 | 备注 |

#### 查询条件

| 条件 | 类型 | 说明 |
|------|------|------|
| invoice_type | int | 进项/销项 |
| invoice_no | string | 发票号 |
| seller/buyer_name | string | 销方/购方名称 |
| invoice_date_range | date[] | 开票日期范围 |
| status | int | 发票状态 |
| tax_type_id | long | 税种（按税率关联） |

#### 接口
- `GET /api/tax/invoice/page` — 发票分页查询
- `GET /api/tax/invoice/{id}` — 发票详情
- `POST /api/tax/invoice` — 登记发票
- `PUT /api/tax/invoice/{id}` — 修改发票信息 |
| `POST /api/tax/invoice/{id}/verify` | — 认证进项发票（标记为已认证）
- `POST /api/tax/invoice/{id}/void` | — 作废销项发票
- `GET /api/tax/invoice/statistics` — 发票统计（按月度进项/销项汇总）
- `POST /api/tax/invoice/import` | — 批量导入发票

### T-05 税务报表

#### 业务规则
- **税负分析表**：各税种应纳税额汇总与趋势
- **增值税明细表**：销项/进项/应纳税额明细
- **企业所得税计算表**：利润总额、纳税调整、应纳税所得额、应纳税额
- **综合税负率**：综合税负 = 总纳税额 / 营业收入 × 100%
- 支持按月度/季度/年度展示
- 支持同比/环比分析

#### 接口
- `GET /api/tax/report/tax-burden` — 税负分析
- `GET /api/tax/report/vat-detail` — 增值税明细
- `GET /api/tax/report/cit-calculation` — 企业所得税计算表
- `GET /api/tax/report/comprehensive-rate` — 综合税负率

### T-06 税务日历

#### 业务规则
- 展示各税种的申报截止日期
- 自动按当前日期标识「即将到期」「已逾期」
- 支持自定义提醒（提前 N 天提醒）
- 逾期未申报的标红提醒

#### 日历数据

| 税种 | 申报周期 | 截止日规则 |
|------|---------|-----------|
| 增值税 | 月度 | 次月15日前 |
| 企业所得税 | 季度 | 季度结束后次月15日前 |
| 个人所得税 | 月度 | 次月15日前 |
| 附加税 | 月度 | 随增值税一同申报 |

#### 接口
- `GET /api/tax/calendar` — 获取税务日历数据（指定年度）
- `PUT /api/tax/calendar/remind-config` — 修改提醒配置
