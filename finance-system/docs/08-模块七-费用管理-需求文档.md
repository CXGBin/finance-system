# 模块七：费用管理 — 详细功能需求文档

> 版本：v1.0 | 日期：2026-06-15 | 状态：待确认  
> 模块标识：`expense` | 默认状态：开启  
> 依赖模块：`account`（账务管理，报销凭证生成）、`system`（系统管理）、`approval`（审批流程，可选）

---

## 1. 模块定位

费用管理模块面向员工日常报销和企业费用管控场景，提供费用在线报销、费用分摊、费用统计和预算对比分析功能。可与审批流程模块联动实现报销审批流程，与账务模块联动自动生成费用凭证。

## 3. 详细功能说明

### E-01 费用类型管理

#### 业务规则
- 费用类型用于对报销单中的费用行进行分类
- 预置常见费用类型：差旅费、交通费、餐饮费、住宿费、办公费、通讯费、招待费、培训费等
- 每个费用类型关联一个默认会计科目（用于自动生成凭证）
- 支持配置单次报销限额和月度累计限额
- 费用类型状态：启用 / 停用

#### 费用类型字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| type_name | string | 是 | 费用类型名称 |
| type_code | string | 是 | 费用类型编码（唯一） |
| default_subject_id | bigint FK | 是 | 默认关联会计科目 |
| single_limit | decimal(18,2) | 否 | 单次报销限额（0=不限） |
| monthly_limit | decimal(18,2) | 否 | 月度累计限额（0=不限） |
| need_invoice | bool | 是 | 是否需要发票 |
| sort_order | int | 否 | 排序 |
| status | tinyint | 是 | 0停用 1启用 |

#### 接口
- `GET /api/expense/type/list` — 费用类型列表
- `POST /api/expense/type` — 新增
- `PUT /api/expense/type/{id}` — 修改
- `DELETE /api/expense/type/{id}` — 删除

---

### E-02 费用报销申请

#### 业务规则
- 员工通过系统提交报销单
- 报销单包含：报销信息（总额、报销事由）+ 多行费用明细
- 每行明细需关联费用类型、填写金额、上传票据附件
- 报销金额自动汇总（各明细金额之和）
- 报销单状态：草稿(0)、待审批(1)、审批中(2)、已通过(3)、已驳回(4)、已付款(5)、已作废(6)
- 草稿状态可修改，提交后不可修改（驳回后可重新编辑再提交）
- 同一笔费用不可重复报销（系统校验）
- 报销金额超过费用类型限额时自动拦截提示

#### 报销单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| claim_no | string | 自动 | 报销单号 |
| claim_title | string | 是 | 报销标题 |
| description | string | 是 | 报销事由 |
| total_amount | decimal(18,2) | 自动 | 报销总额（明细汇总） |
| applicant_id | bigint FK | 自动 | 申请人 |
| dept_id | bigint FK | 自动 | 申请人部门 |
| status | int | 自动 | 状态 |
| items | object[] | 是 | 费用明细（至少1条） |

#### 费用明细字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| expense_type_id | bigint FK | 是 | 费用类型 |
| amount | decimal(18,2) | 是 | 报销金额 |
| occur_date | date | 是 | 费用发生日期 |
| description | string | 否 | 费用说明 |
| invoice_no | string | 条件 | 发票号（需发票时必填） |
| attachments | string[] | 条件 | 附件URL（需发票时必填） |

#### 接口
- `GET /api/expense/claim/page` — 报销单分页（支持按状态、申请人、部门、时间范围筛选）
- `GET /api/expense/claim/{id}` — 报销单详情（含明细和审批记录）
- `POST /api/expense/claim` — 新增报销单（草稿）
- `PUT /api/expense/claim/{id}` — 修改草稿
- `POST /api/expense/claim/{id}/submit` — 提交审批
- `POST /api/expense/claim/{id}/cancel` — 撤回报销单（仅待审批状态）
- `POST /api/expense/claim/{id}/void` — 作废报销单

### E-03 费用明细管理

#### 业务规则
- 明细依附于报销单，不可单独操作
- 新增报销单时同时录入明细
- 修改草稿时可增删改明细行
- 明细中的附件支持多文件上传（图片/PDF）

#### 接口
- 明细操作通过报销单接口一并完成（`items` 字段）

---

### E-04 报销审批流程

#### 业务规则
- 报销单提交后自动匹配审批模板（业务类型：`expense_claim`）
- 若审批模块未开启，则具有权限的管理员直接审批
- 审批通过后报销单状态变为「已通过」
- 审批驳回后申请人可修改后重新提交
- 审批过程记录完整轨迹（通过审批模块记录）

#### 接口
- `POST /api/expense/claim/{id}/approve` — 审批通过
- `POST /api/expense/claim/{id}/reject` — 审批驳回

---

### E-05 报销付款

#### 业务规则
- 审批通过的报销单进入付款环节
- 财务人员确认付款（线下转账后系统标记）
- 付款确认时记录：付款日期、付款方式（银行转账/现金）、付款备注
- 付款完成后自动生成会计凭证：
  - 借：管理费用-XX费用
  - 贷：银行存款 / 应付账款

#### 付款字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| payment_date | date | 是 | 付款日期 |
| payment_method | int | 是 | 1银行转账 2现金 3其他 |
| bank_account | string | 否 | 付款银行账号 |
| remark | string | 否 | 备注 |

#### 接口
- `POST /api/expense/claim/{id}/pay` — 确认付款
- `GET /api/expense/claim/pending-payment` — 待付款列表

---

### E-06 费用分摊

#### 业务规则
- 公共费用（如水电费、房租等）按规则分摊到多个部门或项目
- 分摊方式：按人数比例 / 按面积比例 / 按自定义比例 / 平均分摊
- 创建分摊单，选择待分摊的费用和分摊目标
- 分摊金额之和必须等于原始费用金额
- 分摊完成后按分摊结果生成各部门的费用凭证

#### 分摊单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| allocate_no | string | 自动 | 分摊单号 |
| source_voucher_id | bigint FK | 是 | 源凭证ID |
| total_amount | decimal(18,2) | 是 | 分摊总额 |
| allocate_method | int | 是 | 1按人数 2按面积 3自定义比例 4平均 |
| description | string | 是 | 分摊说明 |
| status | int | 是 | 0草稿 1已确认 2已生成凭证 |

#### 分摊明细字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| target_dept_id | bigint FK | 是 | 分摊目标部门 |
| allocate_ratio | decimal(5,2) | 是 | 分摊比例(%) |
| allocate_amount | decimal(18,2) | 是 | 分摊金额 |

#### 接口
- `GET /api/expense/allocate/page` — 分摊记录分页
- `POST /api/expense/allocate` — 创建分摊单
- `PUT /api/expense/allocate/{id}` — 修改分摊单 |
| `POST /api/expense/allocate/{id}/confirm` | — 确认分摊
- `POST /api/expense/allocate/{id}/generate-voucher` | — 生成分摊凭证

---

### E-07 费用统计

#### 业务规则
- 多维度费用统计：
  - **按部门**：各部门费用支出汇总
  - **按费用类型**：各类费用占比
  - **按期间**：月度/季度/年度费用趋势
  - **按人员**：个人费用报销排名
- 支持图表展示（柱状图、饼图、折线图）

#### 接口
- `GET /api/expense/statistics/by-dept` — 按部门统计
- `GET /api/expense/statistics/by-type` — 按费用类型统计
- `GET /api/expense/statistics/by-period` — 按期间趋势
- `GET /api/expense/statistics/by-person` — 按人员统计

---

### E-08 预算对比

#### 业务规则
- 仅在预算模块开启时可用
- 实时对比各部门/各费用类型的实际支出与预算
- 展示：预算金额、已支出金额、剩余预算、执行率
- 超预算项目标红预警

#### 接口
- `GET /api/expense/budget-compare` — 费用预算对比

---

### E-09 费用报表导出

#### 业务规则
- 支持报销明细表、费用统计表导出
- 格式：Excel、PDF

#### 接口
- `GET /api/expense/export/claims` — 导出报销明细
- `GET /api/expense/export/statistics` — 导出费用统计
